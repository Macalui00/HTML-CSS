El color permite darle color a cualquier parte o elemento del HTML vamos a aplicar colores de muchas formas, entre ellas Valores por nombre del color --> p {color: red;} valores en notacion hexadecimal --> p {color:#FF0000;} (numeral, el valor del canal rojo, del canal verde y del canal azul) valores en notacion hexagecimal abreviada --> p{color: #F00;} valores en notacion RGB --> p {color:rgb(255,0,0)} (donde tengo que especificar que el color es de tipo RGB y entre parentesis el color de cada canal) esos canales tienen unos valores donde van de 0 a 255, 0 ausencia de color, 255 saturacion del color. ausencia de color: negro. saturacion del color, blanco.
